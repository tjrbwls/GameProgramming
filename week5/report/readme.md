## 8_5_1 Upgrade
업그레이드된 점
- 게임 실행 시 테두리가 깨지는 현상 수정
- 말과 화살이 겹치는 현상 수정
- 화살이 목표를 맞출 시 그에 따른 텍스처 출력과 비프음 출력 기능 추가
- 화살이 목표를 못 맞출 시 그에 따른 텍스처 출력
- 결과 창 개선 및 게임의 진행 여부 선택 가능하게 한 기능 추가
- 최고 플레이어 및 기록 표시 기능 추가
- 시도 횟수 및 난이도 조절 기능 추가
- 말 모양 변경 기능 추가
